# README

## 概述

该插件是为OnlyOffice开发的一款定制化、创新性的内容增强工具，旨在为用户提供一种便捷、有趣的方式来丰富其文档、表格和演示文稿的内容。不同于传统的文字表达，这款插件允许用户插入图片、手绘图案，以及自由组合表情包并插入，使文档更加生动、个性化和易于理解。
